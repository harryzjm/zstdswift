# ZstdSwift

  swift category for zstandard compression.

## Requirements

- iOS 13.0+
- Xcode 15.0+
- Swift 5.0

## Installation

### CocoaPods

```
pod 'ZstdSwift'
```

## Usage

```swift
public extension Data {
    func compressZstd(_ level: Int32 = ZSTD_CLEVEL_DEFAULT) throws -> Data
    func decompressZstd() throws -> Data
}
```
